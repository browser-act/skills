# Comprehensive SEO Analyzer

This skill performs end-to-end SEO analysis combining on-page, technical, and content quality audits into a single comprehensive report. It provides a holistic view of your SEO health with prioritized action items.

## When to Use This Skill

- Complete SEO health check for websites or web applications
- Pre-launch SEO validation
- Quarterly or monthly SEO audits
- Identifying why a site isn't performing well
- Creating comprehensive SEO strategy
- Client SEO reports
- Competitive SEO analysis
- Post-migration validation

## What This Skill Does

1. **Meta Tags Analysis**: Title, description, Open Graph, Twitter Cards, canonical tags
2. **Content Structure Audit**: Heading hierarchy, keyword usage, internal linking, readability
3. **Technical SEO Check**: URLs, robots.txt, sitemap, Core Web Vitals, HTTPS, structured data
4. **Performance Optimization**: Page speed, resource loading, caching, render-blocking resources
5. **Accessibility Review**: ARIA labels, color contrast, keyboard navigation, screen reader compatibility
6. **Mobile SEO Assessment**: Mobile-first indexing, responsive design, touch targets
7. **Content Quality Evaluation**: E-E-A-T, originality, user intent, freshness
8. **Local SEO Check**: NAP consistency, local schema markup (when applicable)

## Best Practices

1. **Be Thorough But Efficient**
   - Focus on high-impact issues first
   - Provide specific, actionable recommendations
   - Include code examples where helpful

2. **Use Tools Effectively**
   - **WebFetch**: Analyze live URLs
   - **Read**: Examine HTML/template files
   - **Grep**: Search for meta tags, schema markup
   - **Bash**: Check file sizes, run validators

3. **Provide Context**
   - Include industry benchmarks
   - Explain why issues matter
   - Estimate impact of fixes

4. **Prioritize Recommendations**
   - Critical issues affecting indexability/crawling
   - Performance issues affecting Core Web Vitals
   - Content issues affecting rankings
   - Enhancement opportunities

5. **Be Specific**
   - Include file paths and line numbers
   - Provide before/after examples
   - Give exact measurements and targets
