# Comprehensive SEO Analyzer

A complete, all-in-one SEO analysis skill that combines on-page, technical, performance, mobile, accessibility, and content quality audits into a single comprehensive report.

## What It Does

This skill performs 8 major types of SEO analysis:

1. **Meta Tags Analysis** - Title, descriptions, OG tags, Twitter Cards
2. **Content Structure** - Headings, keywords, internal links, images
3. **Technical SEO** - URLs, robots.txt, sitemap, schema, indexability
4. **Performance** - Core Web Vitals, page speed, resource optimization
5. **Mobile SEO** - Mobile-friendliness, responsive design, touch targets
6. **Accessibility** - WCAG compliance, ARIA, keyboard navigation
7. **Content Quality** - E-E-A-T, originality, user intent, freshness
8. **Local SEO** - NAP consistency, local schema (when applicable)

## When to Use

- Complete SEO health checks
- Pre-launch validation
- Monthly/quarterly audits
- Diagnosing performance issues
- Client reporting
- Competitive analysis

## Quick Start

```
Perform a comprehensive SEO analysis for https://example.com
```

```
Full SEO audit for this Next.js application
```

```
Compare SEO between [your URL] and [competitor URL]
```

## What You Get

- **Overall SEO Score** out of 100
- **Critical Issues** requiring immediate attention
- **Quick Wins** for fast improvements
- **Detailed Recommendations** with code examples
- **Action Plan** with prioritized tasks
- **Competitive Comparison** against top sites
- **Expected Results** from implementing fixes

## Output Format

The skill generates a comprehensive report with:

- Executive summary with overall score
- Score breakdown by category (8 areas)
- Critical/Important/Minor issues prioritized
- Specific fixes with file locations and line numbers
- Before/after recommendations
- Actionable checklist
- Timeline for implementation
